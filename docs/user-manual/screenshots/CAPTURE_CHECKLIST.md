# LeagueSphere Screenshot Capture Checklist

**Target Site:** https://leaguesphere.app/
**Resolution:** 1440x900
**Format:** PNG
**Browser:** Chrome (recommended)

## Pre-Capture Setup

1. Open Chrome browser
2. Set viewport size to 1440x900:
   - Press F12 to open DevTools
   - Click "Toggle device toolbar" (Ctrl+Shift+M)
   - Set dimensions to 1440x900
   - Close DevTools (F12) for clean screenshots
3. Navigate to https://leaguesphere.app/
4. Zoom level: 100%

## Post-Capture Tasks

### Image Optimization

After capturing all screenshots, optimize them:

```bash
cd /home/cda/dev/leaguesphere/docs/user-manual/screenshots/

# Using optipng (lossless compression)
find . -name "*.png" -exec optipng -o5 {} \;

# OR using pngquant (lossy but better compression)
find . -name "*.png" -exec pngquant --quality=85-95 --ext .png --force {} \;

# Check file sizes
find . -name "*.png" -exec ls -lh {} \;
```

### Verify Capture

```bash
# Count screenshots per directory
for dir in intro gamedays passcheck scorecard liveticker league_table officials teammanager accounts; do
  echo "$dir: $(ls -1 $dir/*.png 2>/dev/null | wc -l) screenshots"
done
```

## Tips for Clean Screenshots

1. **Clear browser cache** before starting to ensure fresh page loads
2. **Disable browser extensions** that might add UI elements
3. **Use incognito mode** for consistent experience
4. **Hide personal information** if needed (use browser DevTools to edit text)
5. **Capture loading states** where relevant (spinners, progress bars)
6. **Capture error states** if you can trigger them safely
7. **Take multiple shots** of complex interfaces (before/after interactions)
8. **Consistent timing** - capture at similar stages (e.g., mid-game for liveticker)

## Notes

- Some pages may require authentication - login with test credentials
- Public pages should be captured first (homepage, login, public standings)
- Private pages may require specific user roles (admin, team manager, etc.)
- If a feature is unavailable, document it in the checklist
