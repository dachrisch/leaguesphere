Liveticker

Überblick

Der Liveticker ist eine Echtzeit-Anzeige für aktuelle Spielereignisse an einem Spieltag. Er ermöglicht es Zuschauern, Trainern und Organisatoren, den Verlauf mehrerer Spiele gleichzeitig zu verfolgen, ohne physisch vor Ort sein zu müssen.

Der Liveticker ist als React-Anwendung mit Redux State Management implementiert und aktualisiert sich automatisch, sobald neue Ereignisse aus der Scorecard eingetragen werden. Er bietet eine übersichtliche Chronologie aller Spielaktionen in Echtzeit.

Hinweis: Der Liveticker ist eine reine Anzeigekomponente. Die eigentlichen Spielereignisse werden über die Scorecard eingetragen. Der Liveticker zeigt diese dann automatisch an.

Liveticker aufrufen

Sie erreichen den Liveticker über das Hauptmenü auf der Homepage:

  1. Öffnen Sie die LeagueSphere-Startseite
  2. Klicken Sie im Navigationsmenü auf "Liveticker"
  3. Der Liveticker öffnet sich und zeigt alle aktiven Spiele des aktuellen Spieltags

Wenn aktuell keine Spiele laufen oder noch keine Ereignisse eingetragen wurden, zeigt der Liveticker eine entsprechende Meldung an.

Funktionsweise

Automatische Aktualisierung

Der Liveticker aktualisiert sich automatisch in regelmäßigen Intervallen (typischerweise alle 30-60 Sekunden), um neue Ereignisse anzuzeigen. Sie müssen die Seite nicht manuell neu laden.

Angezeigte Ereignisse

Der Liveticker zeigt folgende Spielereignisse in chronologischer Reihenfolge an:

  • Touchdowns: Anzeige des Punktestands nach einem erfolgreichen Touchdown (6 Punkte)
  • Extra Points: Zusätzliche Punkte nach Touchdowns (1 oder 2 Punkte)
  • Safety: 2 Punkte für die verteidigende Mannschaft
  • Spielbeginn: Start des Spiels mit Anstoßzeit
  • Halbzeit: Beginn der Halbzeitpause
  • Spielende: Abschluss des Spiels mit Endergebnis

Ereignisformat

Jedes Ereignis wird mit folgenden Informationen angezeigt:

  • Zeitstempel: Wann das Ereignis eingetreten ist (z.B. "14:35 Uhr")
  • Spielzeit: Aktuelle Spielzeit (z.B. "Q2 08:45" für 2. Quarter, 8:45 verbleibend)
  • Teams: Beteiligte Mannschaften
  • Ereignisbeschreibung: Details zum Spielzug
  • Aktueller Spielstand: Punktestand nach dem Ereignis

Ansichtsoptionen

Mehrere Spiele gleichzeitig

Wenn mehrere Spiele gleichzeitig stattfinden, zeigt der Liveticker alle aktiven Spiele in einer gemeinsamen Chronologie an. Neueste Ereignisse erscheinen oben in der Liste.

Spielfilterung

In einigen Ansichten können Sie den Liveticker auf ein einzelnes Spiel filtern:

  • Klicken Sie auf einen Spielnamen in der Übersicht
  • Der Liveticker zeigt nun nur Ereignisse dieses Spiels
  • Über "Alle Spiele anzeigen" kehren Sie zur Gesamtansicht zurück

Historische Ansicht

Der Liveticker zeigt standardmäßig nur Ereignisse des aktuellen Spieltags. Ältere Ereignisse werden nicht angezeigt, können aber über die Spieltagsdetails eingesehen werden.

Anwendungsfälle

Für Zuschauer

Zuschauer können den Liveticker nutzen, um:

  • Den Verlauf von Spielen zu verfolgen, bei denen sie nicht physisch anwesend sind
  • Parallel mehrere Spiele auf verschiedenen Feldern im Blick zu behalten
  • Spielergebnisse in Echtzeit zu erfahren
  • Wichtige Spielmomente nachzuvollziehen

Für Trainer und Teams

Trainer können den Liveticker nutzen, um:

  • Ergebnisse von Konkurrenzspielen zu verfolgen
  • Die Tabellenposition während des Spieltags zu beobachten
  • Spielverläufe zu analysieren

Für Organisatoren

Ligaorganisatoren nutzen den Liveticker für:

  • Überwachung aller laufenden Spiele an einem Spieltag
  • Schnelle Erkennung von Auffälligkeiten oder Problemen
  • Koordination zwischen verschiedenen Spielfeldern
  • Öffentliche Kommunikation von Spielständen

Technische Details

Architektur

Der Liveticker ist als eigenständige React-Anwendung implementiert:

  • Frontend: React mit Redux State Management
  • Backend: Django REST Framework API unter /api/liveticker/
  • Aktualisierung: Polling-Mechanismus (regelmäßige API-Abfragen)
  • Datenquelle: Ereignisse aus der Scorecard-Anwendung

API-Endpoints

Der Liveticker kommuniziert mit folgenden Backend-Endpoints:

  • GET /api/liveticker/events/ - Abruf aller aktuellen Ereignisse
  • GET /api/liveticker/games/ - Liste aktiver Spiele
Hinweis für Entwickler: Die Liveticker-Anwendung wird mit webpack gebaut und in liveticker/static/liveticker/js/liveticker.js ausgegeben. Nach Änderungen am Frontend-Code muss npm --prefix liveticker/ run build ausgeführt werden.

Best Practices

Als Zuschauer

  • Automatische Aktualisierung: Lassen Sie die Seite einfach geöffnet - sie aktualisiert sich automatisch
  • Mobile Nutzung: Der Liveticker ist responsive und funktioniert gut auf Smartphones
  • Netzwerkverbindung: Stellen Sie sicher, dass Sie eine stabile Internetverbindung haben
  • Browser-Tabs: Sie können mehrere Liveticker-Tabs öffnen für verschiedene Ligen oder Spieltage

Als Organisator

  • Öffentliche Anzeige: Projizieren Sie den Liveticker auf eine Leinwand für Zuschauer vor Ort
  • Social Media: Teilen Sie den Liveticker-Link für Live-Updates während des Spieltags
  • Qualitätskontrolle: Überprüfen Sie den Liveticker, um sicherzustellen, dass alle Spiele korrekt erfasst werden
  • Backup-Anzeige: Nutzen Sie den Liveticker als Backup, falls die Hauptanzeige ausfällt

Fehlerbehebung

Keine Ereignisse werden angezeigt

Mögliche Ursachen und Lösungen:

  • Keine aktiven Spiele: Überprüfen Sie, ob aktuell Spiele laufen und ob in der Scorecard Ereignisse eingetragen werden
  • Netzwerkproblem: Prüfen Sie Ihre Internetverbindung und laden Sie die Seite neu
  • Browser-Cache: Löschen Sie den Browser-Cache und laden Sie die Seite erneut (Strg+F5)
  • Falscher Spieltag: Stellen Sie sicher, dass Sie den aktuellen Spieltag ausgewählt haben

Ereignisse erscheinen nicht in Echtzeit

Mögliche Ursachen:

  • Das Aktualisierungsintervall liegt bei 30-60 Sekunden - es kann eine kurze Verzögerung geben
  • Bei mobilen Geräten kann das Polling pausieren, wenn der Browser in den Hintergrund wechselt
  • Netzwerklatenz kann zu Verzögerungen führen

Lösung: Laden Sie die Seite manuell neu, um sofort die neuesten Ereignisse zu sehen.

Falsche oder veraltete Informationen

Wenn der Liveticker falsche Informationen anzeigt:

  1. Laden Sie die Seite neu (F5)
  2. Überprüfen Sie die Spieltagsdetails für die korrekten Informationen
  3. Kontaktieren Sie den Spieltag-Organisator, falls Korrekturen nötig sind
  4. Korrekturen werden über die Scorecard vorgenommen

Darstellungsprobleme

Bei Layout- oder Darstellungsproblemen:

  • Nutzen Sie einen modernen Browser (Chrome, Firefox, Safari, Edge)
  • Stellen Sie sicher, dass JavaScript aktiviert ist
  • Deaktivieren Sie Browser-Erweiterungen, die Inhalte blockieren könnten
  • Versuchen Sie es mit einem anderen Gerät oder Browser

Zusammenfassung

Der Liveticker ist ein leistungsstarkes Werkzeug für Echtzeit-Updates während eines Spieltags:

  • Automatische Aktualisierung aller Spielereignisse
  • Übersichtliche Chronologie mit Zeitstempeln und Spielständen
  • Unterstützung für mehrere gleichzeitige Spiele
  • Responsive Design für Desktop und Mobile
  • Keine Anmeldung erforderlich (öffentlich zugänglich)
Tipp: Kombinieren Sie den Liveticker mit der Ligatabelle, um sowohl aktuelle Spielereignisse als auch den Gesamtstand der Liga im Blick zu behalten.
